# KMVision-1: Synthetic Clinical Chart Dataset Pipeline

This project provides a robust, parallelized Python pipeline to programmatically generate hundreds of thousands of realistic synthetic clinical charts (Kaplan-Meier curves, Forest plots, Waterfall plots, and various Anchor charts). 

Every generated chart is automatically mapped to perfectly matched JSON ground-truth schemas detailing coordinate positions and survival variables. The resulting paired dataset is specifically engineered to bridge the sim-to-real gap and train Vision-Language Models (VLMs) for clinical and academic chart data extraction.
